# Strands Agents MCP Server

Documentation server for Strands Agents SDK, enabling AI assistants to help build AI agents using the Strands Agents framework.

## Prerequisites

- **uv package manager** installed (`brew install uv` or see [uv docs](https://docs.astral.sh/uv/getting-started/installation/))

## Setup

No additional setup required. The server provides documentation about Strands Agents SDK.

## Usage

Once configured, AI assistants can:
- Access Strands Agents SDK documentation
- Get help building AI agents
- Understand agent patterns and workflows

## Configuration

No API keys required. Uses `uvx strands-agents-mcp-server`.
