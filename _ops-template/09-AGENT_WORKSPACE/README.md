# 09 · Agent Workspace

All AI agent work-product lives here. Code stays in the app repo; plans, handovers, decisions, and notes go here.

| Folder | Holds |
|---|---|
| `plans/<N>-<slug>/` | Per-issue PRD.md / research.md / implementation-plan.md |
| `handovers/` | Per-run handover summaries: `YYYY-MM-DD-<agent>-<scope>.md` |
| `decisions/` | Decision records (what was chosen, why, alternatives) |
| `notes/` | Scratch research, working notes |

Every doc written here gets a row in `../INDEX.md`.
